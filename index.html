CSS Grid

  • It's not an another framework
  • It's not a library

  • It's simply display: grid;

Why CSS Grid?

  • Powerful 2-D Layouting with rows & columns
  • No More Floats
  • No More Positioning with 'position'
  • Very neat HTML: no more class names for layouting

  • Download Size: 0.000Kb

Getting Started

  • CSS Grid Layout is applied with reference to:
    • A parent element with: display: grid; or display: inline-grid;
    • One or more direct child elements
  • CSS properties can be applied on the parent element and child elements as well

The Grid Template

Size of each column and row can be defined in the grid-template

Sizes can be defined as:

  • Length/Percentage: px, rem, cm, vw/vh, %
  • Auto: Auto length of child element
  • fr: fraction of remaining space
